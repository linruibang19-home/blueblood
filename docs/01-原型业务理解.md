# 01-原型业务理解

## 文档用途

本文件用于帮助开发者和 AI IDE 理解原型背后的业务，不直接要求写代码。

## 产品本质

蓝血菁英平台不是单一社区、课程平台或外包平台，而是一个“超级个体成长与商业任务交付平台”。

它将以下能力组合在一起：

```text
社区发现
兴趣小组
人才展示
课程成长
作业批改
黑客松组队
岗位机会
任务大厅
任务交付
里程碑审核
收益结算
```

## 前台四大主线

### 发现

用于用户发现平台内容、加入兴趣小组、查看热门任务、认识超级个体。

主要对象：

```text
小组
帖子
用户
热门任务
活动
私信
```

### 成长

用于提升用户能力和等级。

主要对象：

```text
课程
章节
学习进度
作业
批改结果
黑客松
岗位机会
```

### 任务

用于商业任务撮合和执行。

主要对象：

```text
任务
任务分类
任务详情
接单
我的任务
里程碑
交付物
```

### 我的

用于展示用户资产、收益、通知和个人设置。

主要对象：

```text
用户资料
等级
积分
信誉评分
余额
待结算
通知
设置
技能标签
```

## 原型必须复原的业务闭环

第一阶段 H5 复原时，至少需要能演示以下闭环：

```text
发现首页 -> 小组详情 -> 帖子详情 -> 回复
发现首页 -> 超级个体详情 -> 建立连接 -> 私信
成长首页 -> 课程详情 -> 视频学习 -> 课堂互动
成长首页 -> 作业提交 -> 批改结果
成长首页 -> 黑客松 -> 创建队伍/提交项目
任务首页 -> 任务详情 -> 确认接单
任务首页 -> 我的任务 -> 任务执行 -> 里程碑提交
我的首页 -> 收益明细
我的首页 -> 消息通知
我的首页 -> 设置 -> 编辑个人资料
```
