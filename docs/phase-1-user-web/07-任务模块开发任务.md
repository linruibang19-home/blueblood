# 07-任务模块开发任务

## 任务目标

复原任务模块，包括任务大厅、任务详情、接单、我的任务、任务执行和里程碑提交。

## 开发范围

允许修改：

```text
apps/user-web/src/pages/task/
apps/user-web/src/router/
apps/user-web/src/components/
apps/user-web/src/stores/
```

必要时可补充：

```text
apps/user-web/src/mock/
apps/user-web/src/api/
```

## 页面清单

创建：

```text
TaskHome.vue
TaskDetail.vue
TaskExecution.vue
MilestoneSubmit.vue
```

## 路由清单

```text
/tasks
/tasks/:id
/task-orders/:id
/task-orders/:id/milestones/:milestoneId/submit
```

## 交互要求

```text
任务大厅/我的任务 Tab 切换
分类筛选任务
点击任务进入详情
点击确认接单弹出确认框
确认后 Toast
点击我的任务进入任务执行
点击提交里程碑进入提交页
提交里程碑后 Toast
```

## 禁止事项

```text
不要实现真实接单接口
不要实现真实文件上传
不要开发后台任务审核
不要开发收益结算逻辑
```

## 验收标准

```text
任务大厅可筛选
任务详情可查看
接单弹窗可操作
我的任务可进入执行页
```
