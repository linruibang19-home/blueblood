# 06-成长模块开发任务

## 任务目标

复原成长模块，包括课程、视频学习、作业、黑客松和岗位。

## 开发范围

允许修改：

```text
apps/user-web/src/pages/grow/
apps/user-web/src/router/
apps/user-web/src/components/
apps/user-web/src/stores/
```

必要时可补充：

```text
apps/user-web/src/mock/
apps/user-web/src/api/
```

## 页面清单

创建：

```text
GrowHome.vue
CourseDetail.vue
VideoStudy.vue
AssignmentSubmit.vue
AssignmentResult.vue
HackathonDetail.vue
JobDetail.vue
```

## 路由清单

```text
/grow
/courses/:id
/courses/:id/video/:chapterId
/assignments/:id/submit
/assignments/:id/result
/hackathons/:id
/jobs/:id
```

## 禁止事项

```text

```

