# 07-钱包收益与通知模块任务

## 任务目标

实现钱包、收益记录、提现记录和通知接口。

## 模块范围

```text
wallet
notification
```

## 接口能力

```text
钱包概要
收益明细
待结算列表
提现记录
创建提现申请
通知列表
通知详情
标记已读
批量已读
创建系统通知预留接口
```

## 前台关联页面

```text
我的首页
收益明细
消息通知
```

## 验收标准

```text
钱包金额字段准确使用 DECIMAL
收益记录可按状态查询
通知可分页
通知可标记已读
```

## 交给 AI IDE 的执行提示

```text
请只开发钱包收益与通知模块。实现钱包概要、收益流水、提现记录、通知列表和已读操作，不要开发任务模块或后台页面。
```
