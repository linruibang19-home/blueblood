# 03-认证与用户模块任务

## 任务目标

实现登录认证、当前用户、个人资料、技能标签、等级积分和认证状态相关接口。

## 模块范围

```text
auth
user
profile
```

## 接口能力

```text
注册/登录
退出登录
获取当前用户
查看用户详情
编辑个人资料
技能标签增删
查看等级积分
查看信誉评分
提交认证申请
管理员审核认证预留接口
```

## 前台关联页面

```text
我的首页
编辑个人资料
超级个体详情
发现首页超级个体列表
```

## 验收标准

```text
接口可通过 Swagger 测试
用户资料可增改查
技能标签可增删
当前用户接口可返回前台所需数据
认证状态字段可流转
```

## 交给 AI IDE 的执行提示

```text
请只开发认证与用户模块接口，包括登录、当前用户、用户资料、技能标签、等级积分和认证状态。不要开发社区、课程或任务模块。
```
