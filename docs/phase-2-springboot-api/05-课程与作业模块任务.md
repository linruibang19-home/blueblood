# 05-课程与作业模块任务

## 任务目标

实现课程、章节、学习进度、作业提交和批改结果接口。

## 模块范围

```text
course
assignment
```

## 接口能力

```text
课程列表
课程详情
章节列表
更新学习进度
获取当前学习章节
作业列表
作业详情
提交作业
查看批改结果
管理员批改作业预留接口
```

## 前台关联页面

```text
成长首页
课程详情
视频学习
作业提交
批改结果
```

## 验收标准

```text
课程可分页查询
章节状态可返回
学习进度可更新
作业提交可保存代码和附件记录
批改结果可查询
```

## 交给 AI IDE 的执行提示

```text
请只开发课程与作业模块接口。实现课程、章节、学习进度、作业提交和批改结果，不要开发任务或财务模块。
```
