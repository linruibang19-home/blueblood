# 08-文件上传与系统配置任务

## 任务目标

实现文件上传、字典配置、系统配置等基础支撑能力。

## 模块范围

```text
file
system
```

## 接口能力

```text
上传文件
查看文件信息
删除文件
字典类型列表
字典项列表
系统配置读取
系统配置更新预留接口
```

## 文件用途

```text
头像
作业附件
里程碑附件
黑客松 PPT
项目附件
```

## 验收标准

```text
文件可保存到本地目录或 MinIO
数据库保存文件记录
返回可访问 URL
字典接口可支撑前台状态显示
```

## 交给 AI IDE 的执行提示

```text
请只开发文件上传与系统配置模块。实现上传、文件记录、字典和配置读取，不要开发业务页面。
```
