# 02-数据库模型设计

## 任务目标

根据原型和第一阶段 Mock 数据设计 MySQL 数据库表结构。

## 开发范围

允许修改：

```text
sql/
server/blueblood-api/src/main/resources/
```

## 数据库规范

```text
MySQL 8
utf8mb4
InnoDB
主键 BIGINT AUTO_INCREMENT
金额 DECIMAL(10,2)
所有核心表包含 created_at、updated_at、deleted_at
状态字段使用 varchar 或 tinyint，并配枚举
```
