# 04-社区模块任务

## 任务目标

实现兴趣小组、帖子、评论、点赞、收藏、活动和简单私信接口。

## 模块范围

```text
group
post
activity
chat
```

## 接口能力

```text
小组列表
小组详情
加入/退出小组
小组帖子列表
帖子详情
发布帖子
评论帖子
点赞/取消点赞
收藏/取消收藏
活动列表
活动报名
会话列表
消息列表
发送消息
```

## 前台关联页面

```text
发现首页
小组详情
帖子详情
私信聊天
```

## 验收标准

```text
小组列表可分页
帖子可按小组和分类查询
评论可追加
点赞状态可切换
私信可保存消息记录
```

## 交给 AI IDE 的执行提示

```text
请只开发社区模块接口，包括小组、帖子、评论、点赞、收藏、活动报名和简单私信。不要开发课程、任务或财务模块。
```
