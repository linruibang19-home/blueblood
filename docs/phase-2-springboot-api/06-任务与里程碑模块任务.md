# 06-任务与里程碑模块任务

## 任务目标

实现任务大厅、任务详情、接单、我的任务、里程碑提交和审核基础接口。

## 模块范围

```text
task
order
milestone
```

## 接口能力

```text
任务分类列表
任务列表
任务详情
任务筛选
确认接单
我的任务列表
任务执行详情
里程碑列表
提交里程碑
查看里程碑审核结果
管理员审核里程碑预留接口
```

## 关键状态

任务状态：

```text
DRAFT
PENDING_REVIEW
APPROVED
RECRUITING
IN_PROGRESS
COMPLETED
CLOSED
```

里程碑状态：

```text
NOT_STARTED
IN_PROGRESS
SUBMITTED
APPROVED
REJECTED
OVERDUE
```

## 前台关联页面

```text
任务首页
任务详情
任务执行
里程碑提交
消息通知
收益明细
```

## 验收标准

```text
任务可按分类筛选
接单后生成 task_order
我的任务可查询
里程碑提交可保存
状态流转合法
```
