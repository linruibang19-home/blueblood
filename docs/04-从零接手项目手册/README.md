# 04 · 从零接手项目手册

> 面向“几天没看项目 / 刚接手仓库 / 面试前需要快速恢复记忆”的系统学习手册。

---

## 这个文件夹解决什么问题？

你现在可能遇到的是：

- 过了几天没看，忘了项目整体是干什么的；
- 打开仓库后目录很多，不知道先看哪里；
- 文档已经很多，但不知道应该按什么顺序恢复；
- 面试前想重新把业务、代码、数据库、Redis 链路串起来；
- 后续想继续开发，但不知道从哪个模块下手。

这个手册的目标就是：**把你当作一个刚接手项目的人，从 0 带你重新掌握蓝血菁英平台。**
