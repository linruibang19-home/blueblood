# 01-后台管理端总说明

## 任务目标

第三阶段建设后台管理端，用于平台运营、审核、配置和结算。

## 启动条件

必须在第二阶段核心后端接口稳定后启动。

## 技术方案

```text
RuoYi-Vue3 或同类若依体系
Vue3
Vite
TypeScript
Element Plus
Spring Boot 3
MyBatis-Plus
MySQL
Redis
```

## 禁止事项

```text
不要改动前台 H5 页面

```
