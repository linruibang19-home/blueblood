## 🎯 使用方式

### 面试前过一遍
1. 先看 [01-项目介绍类.md](01-项目介绍类.md) — 准备自我介绍
2. 再看 [02-技术亮点类.md](02-技术亮点类.md) — 准备技术亮点
3. 然后看 [03-原理追问类.md](03-原理追问类.md) — 准备追问
4. 最后看 [04-方案设计类.md](04-方案设计类.md) — 展示你有架构思维

### 遇到问题查一下
每个问题都有：
- **题目**：面试官会怎么问
- **标准回答**：可以直接用的模板
- **追问预案**：面试官可能的追问

## ⚠️ 注意事项

1. **不要背**：理解原理比背答案更重要
2. **结合项目**：每个答案都要联系项目
3. **展示思考**：不仅要会说结论，还要说"为什么这么想"
4. **诚实**：不会就说不会，不要瞎编
