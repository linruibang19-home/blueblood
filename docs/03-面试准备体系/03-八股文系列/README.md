## 📊 每篇文档结构

每篇八股文都包含：

1. **核心概念**：是什么？为什么？
2. **项目中的实现**：怎么用的？代码在哪？
3. **面试标准回答**：可以直接用的模板
4. **追问预案**：面试官可能的追问

## ⚠️ 学习方法

1. **先理解再背**：理解原理比背答案更重要
2. **结合项目**：每一条都想想"项目里怎么用的"
3. **准备追问**：面试官会追问细节
4. **动手验证**：有条件的话，写代码验证
