## 🎯 阅读顺序

### 第一次学习（按这个顺序）

```
1. 01-业务全貌与核心闭环     → 先搞清楚业务
2. 03-高并发与稳定性体系(上)  → 技术亮点（重点！）
3. 04-高并发与稳定性体系(下)  → 技术亮点（继续）
4. 05-认证授权体系           → 认证授权
5. 06-核心业务逻辑详解       → 业务细节
6. 02-技术架构深度解析       → 整体架构（回头看）
7. 07-工程规范与设计模式     → 工程规范
```

### 面试前复习（按这个顺序）

```
1. 03-高并发与稳定性体系(上)  → 秒杀是必问
2. 04-高并发与稳定性体系(下)  → 限流/缓存/幂等
3. 05-认证授权体系           → JWT 是常问
4. 01-业务全貌与核心闭环     → 项目介绍用
5. 06-核心业务逻辑详解       → 业务细节
```
