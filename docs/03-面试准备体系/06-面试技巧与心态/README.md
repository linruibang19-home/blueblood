## ⚠️ 核心心态

1. **不要慌**：面试是双向选择，不卑不亢
2. **不要装**：不会就说不会，不要瞎编
3. **展示思考**：不仅要会说结论，还要说"为什么这么想"
4. **保持真诚**：项目是 AI 辅助做的没问题，但要能讲清楚"我理解了"
