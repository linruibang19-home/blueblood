## ⚠️ 注意事项

1. **一页纸**：应届生一页就够了，不要超过两页
2. **量化成果**：用数字说话，不要用"负责"、"参与"
3. **突出亮点**：项目最大的亮点要突出
4. **真实性**：不要虚构，所有内容都能讲清楚
