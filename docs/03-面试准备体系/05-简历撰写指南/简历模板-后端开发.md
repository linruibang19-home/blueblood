## 简历各部分写法

### 1. 基本信息

```markdown
姓名 | 手机号 | 邮箱 | 城市
求职意向：Java 后端开发
```

**注意**：
- 手机号用 3-4-4 格式：138-1234-5678
- 邮箱不要用 QQ 邮箱，用 163/gmail
- 城市写你所在或想去的城市

### 2. 教育背景

```markdown
XX大学 · 计算机科学与技术 · 本科 · 2021-2025
```

**可选加分项**（不要硬凑）：
- GPA：专业前 20% 可以写
- 奖学金：三好学生/奖学金
- 相关课程：数据结构、操作系统、计算机网络（挑跟职位相关的）
- 竞赛：ACM、算法竞赛（拿奖才写）

### 3. 项目经历（重点！）

**STAR 法则**：
- **S（Situation）**：项目背景
- **T（Task）**：你的任务
- **A（Action）**：你做了什么（技术动作）
- **R（Result）**：结果是什么（量化）

**写法示例**：

❌ 不好（太笼统）：
```
负责后端开发，用 Redis 做缓存
```

✅ 好（具体 + 量化）：
```
构建高并发任务抢单系统：Redis Lua 预扣库存 + DB 原子 SQL 兜底，
解决超卖问题；设计 try-catch 回补机制，保证 Redis/DB 最终一致。
单机 QPS 提升 10 倍，从 DB 的 1000 降到 Redis 的 10000。
```

### 4. 技能清单

**格式**：
```
编程语言：Java（熟悉）、Python（了解）
后端框架：Spring Boot、Spring MVC、Spring Security
数据库：MySQL、Redis
中间件：Docker、RabbitMQ
工具：Git、Maven、IDEA、Linux
```

**注意**：
- 不要写"精通"，写"熟悉"或"了解"
- 按掌握程度排序
- 跟职位无关的不要写

### 5. 其他（可选）

**可以写的**：
- 校园经历（跟技术相关的）
- 荣誉奖励（奖学金、竞赛奖）
- 个人项目（有亮点的）

**不要写的**：
- 自我评价（太主观）
- 身份证号
- 生日
- 照片（除非公司要求）
