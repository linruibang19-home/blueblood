# 00-项目总览与开发规则

## 文档用途

本文件是后续交给 AI IDE 或开发者执行任务前必须阅读的总控文档。

开发者不要一次性开发完整系统，应按 `phase-*` 目录中的任务文档逐个执行。每次只处理一个任务文档，完成、验证后再进入下一个任务。

## 项目定位

蓝血菁英平台是面向 AI 技术人才、学生、自由职业者和企业需求方的成长型任务撮合平台。

核心闭环：

```text
用户资料 -> 社区连接 -> 学习成长 -> 任务接单 -> 里程碑交付 -> 审核结算 -> 等级提升
```

## 已确认技术路线

```text
第一阶段：Vue3 + Vite + TypeScript + Vant 复原前台 H5 原型
第二阶段：Spring Boot 3 + MyBatis-Plus + MySQL + Redis 建真实后端
第三阶段：RuoYi-Vue3 / Element Plus 建后台管理端
第四阶段：uni-app + Vue3 适配小程序端
```

## 当前优先级

当前优先执行第一阶段：前台 H5 原型复原。

第一阶段只做：

```text
Vue 前台工程
页面复原
组件拆分
Mock 数据
前端路由
基础 Docker 运行
```

第一阶段不要做：

```text
Spring Boot 后端
真实数据库
后台管理端
小程序端
支付
提现
真实 IM
真实视频流
真实文件上传
```

## AI IDE 执行规则

每次给 AI IDE 的输入应包含：

```text
当前任务文档
必要的上游文档
原型文件路径
明确说明不得跨任务开发
```

每次执行时必须遵守：

```text
只完成当前任务文档要求
不要提前实现后续阶段能力
不要大范围重构无关文件
不要删除原型文件
不要把所有页面堆进一个 Vue 文件
不要把页面内容硬编码在模板里
```

## 原型文件位置

```text
prototype/蓝血精英平台-原型-v6.1.html
```

## 交付判断

每个任务文档都有自己的验收标准。只有当前任务验收通过后，才进入下一个任务。
