# 03-全局布局与基础组件任务

## 任务目标

实现移动端 App 化外壳、底部 Tab、顶部返回栏和基础通用组件。

## 开发范围

允许修改：

```text
apps/user-web/src/layouts/
apps/user-web/src/components/
apps/user-web/src/styles/
apps/user-web/src/router/
apps/user-web/src/pages/
```

## 前置依赖

必须完成：

```text
phase-1-user-web/02-工程初始化任务.md
```

## 布局组件

创建：

```text
MobileTabLayout.vue
SubPageLayout.vue
```

### MobileTabLayout

要求：

```text
最大宽度 430px
居中显示
深色背景
底部固定 Tab
内容区域可滚动
适配安全区
```

底部 Tab：

```text
发现
成长
任务
我的
```

### SubPageLayout

要求：

```text
顶部固定返回栏
标题居中
左侧返回按钮
内容区域可滚动
可选择隐藏底部 Tab
```

