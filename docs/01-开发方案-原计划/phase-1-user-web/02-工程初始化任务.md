# 02-工程初始化任务

## 任务目标

创建前台 H5 Vue 工程，并完成基础依赖、目录、路由、状态管理和样式配置。

## 开发范围

只允许创建或修改：

```text
apps/user-web/
```

## 前置依赖

已阅读：

```text
docs/00-项目总览与开发规则.md
docs/01-原型业务理解.md
docs/02-技术架构与Docker运行方案.md
docs/phase-1-user-web/01-前台H5开发总说明.md
```

## 具体开发内容

创建 Vue 工程：

```text
Vue3
Vite
TypeScript
```

安装依赖：

```text
vant
vue-router
pinia
axios
sass
```

创建目录：

```text
src/router
src/stores
src/api
src/mock
src/layouts
src/components
src/pages
src/styles
```

创建基础文件：

```text
src/router/index.ts
src/router/routes.ts
src/stores/user.ts
src/stores/ui.ts
src/api/request.ts
src/styles/variables.scss
src/styles/base.scss
src/styles/theme.scss
```

配置：

```text
路径别名 @
Vant 按需或全局引入
Pinia
Vue Router
全局基础样式
移动端 viewport 适配
```

## 初始路由

创建以下空页面路由：

```text
/discover
/grow
/tasks
/mine
```

默认重定向：

```text
/ -> /discover
```

## 禁止事项

```text
不要开发具体业务页面
不要开发后端接口
不要写大量 Mock 数据
不要引入复杂 UI 框架替代 Vant
```

## 验收标准

```text
npm install 成功
npm run dev 成功
浏览器打开后进入 /discover
四个空页面路由可以访问
项目目录符合要求
TypeScript 无明显报错
```
