# 01-前台H5开发总说明

## 任务目标

第一阶段使用 Vue3 + Vite + TypeScript + Vant 复原客户提供的移动端 H5 原型。

目标是让客户可以在浏览器中看到一个高度接近原型、可点击、可演示的前台用户端。

## 开发范围

只开发：

```text
apps/user-web/
```

可以新增：

```text
deploy/docker-compose.local.yml
```

不得开发：

```text
Spring Boot 后端
后台管理端
小程序端
真实数据库
真实支付/提现
```

## 原型参考

```text
prototype/蓝血精英平台-原型-v6.1.html
```

## 开发原则

```text
先搭工程，再做布局，再做组件，再做 Mock，再做业务模块
每个页面使用 Mock 数据驱动
不要直接复制原 HTML 到一个 Vue 文件
不要在模板中硬编码所有业务内容
路由、状态、数据、组件要分层
移动端最大宽度按 430px 处理
```

## 验收标准

```text
npm run dev 可以运行
底部四个 Tab 可切换
所有主要子页面可进入和返回
主要交互有 Toast 或状态变化
页面视觉接近原型
Mock 数据集中管理
```
