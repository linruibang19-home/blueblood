# 01-后端开发总说明

## 任务目标

第二阶段使用 Spring Boot 3 + MyBatis-Plus + MySQL + Redis 建设真实后端 API。

## 启动条件

必须在第一阶段前台 H5 验收通过后启动。

## 技术栈

```text
Java 17 或 Java 21
Spring Boot 3
MyBatis-Plus
MySQL 8
Redis 7
Sa-Token 或 Spring Security JWT
Knife4j / Springdoc OpenAPI
Lombok
MapStruct
Hibernate Validator
```

推荐：

```text
Spring Boot 3 + MyBatis-Plus + Sa-Token
```

## 开发范围

只开发：

```text
server/blueblood-api/
sql/
deploy/
```

## 基础能力

必须先建立：

```text
统一响应
统一异常
参数校验
分页返回
登录认证
权限注解
操作日志预留
MySQL 连接
Redis 连接
Swagger/Knife4j 文档
```

## 禁止事项

```text
不要开发后台前端页面
不要开发小程序
不要改变第一阶段前台页面结构
不要一次性实现所有复杂流程
```

## 交给 AI IDE 的执行提示

```text
请只开发 Spring Boot 后端 API。先建立基础工程、统一响应、认证、数据库连接和模块结构，不要开发后台前端或小程序。
```
