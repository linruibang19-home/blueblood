# 01 · 项目总览

## 一句话定位

**蓝血菁英平台** —— 面向 AI 人才的「成长 + 变现」一体化平台:学课程、接任务、参赛/求职、赚收益,形成"学习 → 实战 → 结算"的闭环。

## 背景

本项目是公司既有「蓝血菁英社区」(Laravel/PHP 技术栈)的 **Java 完整重构版本**。重构目标:

- 技术栈升级到 Spring Boot + Vue3 全栈主流方案
- 拆分为 4 端(前台 H5 / 后台 / 小程序 / 后端 API),前后端分离
- 补齐原版缺失的"任务撮合 + 分阶段结算"商业化闭环
- Docker 容器化,一键部署

## 四端全景

| 端 | 目录 | 技术栈 | 职责 |
|----|------|--------|------|
| **后端 API** | `server/blueblood-api` | Spring Boot 3.3 + Java 17 + MyBatis-Plus + MySQL 8 + Redis | 统一业务接口、JWT 鉴权、12 个业务模块 |
| **前台 H5** | `apps/user-web` | Vue3 + Vite + TS + Vant 4 + Pinia | 用户端:发现/成长/任务/我的 |
| **后台管理** | `apps/admin-web` | Vue3 + Vite + TS + Element Plus | 运营端:6 大组 18 个管理页 |
| **微信小程序** | `apps/miniapp` | uni-app + Vue3 + TS | 移动端轻量入口(任务/钱包/通知) |

## 技术栈一览

### 后端
- **框架**:Spring Boot 3.3.5 / Java 17 / Maven
- **ORM**:MyBatis-Plus 3.5.7(分页、自动填充、Lambda 查询)
- **安全**:Spring Security + JWT(jjwt 0.12.6),无状态 + 方法级 RBAC
- **存储**:MySQL 8(主数据) + Redis 7(验证码/缓存)
- **文档**:Knife4j 4.5(OpenAPI 3)
- **工具**:Lombok / MapStruct / Bean Validation
- **规范**:统一响应 `Result<T>` / 全局异常 / 软删除 / 分页 `PageResult`

### 前端
- **框架**:Vue 3 + TypeScript + Vite
- **状态/路由**:Pinia + Vue Router
- **UI**:Vant 4(H5)/ Element Plus(后台)/ uni-app 组件(小程序)
- **请求**:Axios(H5/后台)/ uni.request(小程序),统一拦截解包 + 401 处理

### 部署
- **容器**:Docker + Docker Compose,5 服务编排
- **服务**:api(:8090) / mysql(:3307) / redis(:6380) / user-web(:3000) / admin-web(:3001)

## 核心业务能力

1. **三方式注册登录**:用户名 / 邮箱(验证码) / 手机号(验证码),Redis 存验证码
2. **两种用户角色**:个人用户 / 企业用户(认证审核后可发布岗位/黑客松/任务)
3. **任务撮合闭环**(核心亮点):发布 → 接单 → 里程碑交付 → 雇主验收 → **分阶段结算到钱包**
4. **成长学习**:课程 + 章节 + 线性进度模型 + 作业
5. **社区**:兴趣小组 / 帖子 / 评论 / 点赞 / 聊天
6. **钱包财务**:余额 / 流水 / 提现申请 / 后台审核
7. **企业活动**:岗位发布 / 黑客松发布 / 后台管理
8. **平台运营**:通知 / 权限(RBAC) / 字典 / 配置 / 操作日志 / 文件上传

## 项目规模(量化)

- **44+ 张数据表**,5 次迁移演进(v1 初始化 → v5 课程种子)
- **12 个后端业务模块**,每个含 entity/mapper/service/controller/dto
- **4 端** 约 50+ 个页面
- **5 个 Docker 容器**一键编排
- **11 门** 示例课程 + 完整任务结算演示数据
