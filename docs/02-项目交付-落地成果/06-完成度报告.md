## 已修复的关键 bug(开发过程)

- 钱包提现驳回不退款(资损)→ REJECTED 分支补回 balance
- 退出登录无效(handleLogout 未清 token)→ 清 token + store + 调后端
- 企业注册 401(注册后无登录态)→ 自动登录再提交申请
- 任务里程碑状态枚举错配(大写 vs 小写)→ 映射统一
- 接单后跳转丢订单 id → 用后端返回 orderId 跳执行页
- 进度计算 REJECTED 计入分母卡死订单 → 分母排除 REJECTED
- 接单 slots 读写非原子(超卖)→ 原子 SQL
- email 注册 email 字段被清空 → register 保留 email
